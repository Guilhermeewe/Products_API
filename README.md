# Products API

API REST simples para gerenciamento de produtos utilizando **Java, Spring Boot, JPA** e banco de dados **H2 em memória**.

## Tecnologias

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 Database

## Funcionalidades

- Criar produto
- Listar todos os produtos
- Buscar produto por ID
- Atualizar produto
- Deletar produto
